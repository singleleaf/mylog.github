Ｖｕｅ基础知识

发表于 2019-11-03 | 分类于 Vue

安装

npm install -g @vue/cli

查看已安装版本

vue –version 或者 vue -V

卸载

npm uninstall @vue/cli -g

新建项目

vue create my-project或ｖｕｅ　ｕｉ

项目启动

npm run serve

打包

npm run build

html部分

  1. 插值表达式：存在闪烁问题，但不会覆盖元素中原本的内容
  2. v-cloak：解决插值表达式闪烁的问题
  3. v-text：没有闪烁问题且覆盖元素中原本的内容，内容当作字符串显示
  4. v-html：没有闪烁问题且覆盖元素中原本的内容，内容当作html显示
  5. v-bind：Vue提供的属性绑定机制 缩写是 :
  6. v-on： Vue提供的事件绑定机制 缩写是 @
  7. v-model：可以实现表单元素和 Model 中数据的双向绑定，只能运用再表单元素中

    js部分

  8. new Vue({ })：创建一个Vue实例
  9. el : ‘ ‘ ：绑定Vue的使用范围DOM（要挂在的元素）
  10. data : { } ：展示的数据，用于定义变量
  11. methods : { } ：调用的方法
  12. this：如果想要获取 data 上的数据，或者 想要调用 methods 中的 方法，必须通过 this.数据属性名 或 this.方法名来进行访问，这里的this，就表示 我们 new 出来的 VM 实例对象
  13. filters : { }：定义私有过滤器

    常用指令。

  14. v-model 双向数据绑定，一般用于表单元素。
  15. v-for 对数组或对象进行循环操作，使用是v-for 不是v-repeat
  16. v-on 用来绑定时间，用法：v-on :时间 = ‘函数’
  17. v-show/v-if 用来显示或隐藏元素，v-show 是通过display实现，v-if 是每次删除后在创建

    事件修饰符

  18. .stop：阻止冒泡
  19. .prevent：阻止默认行为
  20. .capture：给元素添加一个监听器，当元素发生冒泡时，先触发带有该修饰符的元素。若有多个该修饰符，则由外而内触发
  21. .self：实现只有点击当前元素时候，才会触发事件处理函数
  22. .once：只触发一次事件处理函数

    键盘事件：@keydown.13 或keydown.enter

    在Vue中使用class，style样式

    直接传递一个数组，这里的class需要使用v-bind做数据绑定：:class=”[‘thin’,’italic’]
    在数组中使用三元表达式：:class=”[‘thin’, ‘italic’, flag?’active’:’’]
    在数组中使用对象代替三元表达式，提高代码可读性：:class=”[‘thin’, ‘italic’, {‘active’:flag}]
    在为class使用v-bind绑定对象的属性是类名，属性的值是个标识符：:class=”{red:true, thin:true, italic:false, active:false}”
  23. vue生命周期
    vue实例从创建到销毁过程成为生命周期

    计算属性

    计算属性也是用来存储数据，具有这俩个特点：数据可以进行逻辑处理操作，对计算属性中的数据进行监视。

    vue实例属性和方法

    属性 vm.$el vm.$data vm.$options vm.$refs
    方法 vm.$mount() vm.$destroy vm.$nextTick(callback) vm.$set(object,key,vlaue) vm.$delete(object,key) vm.$watch(data,callback)

    自定义指令

    自定义全局指令 vue.directive （指令id，定义对象）

    过度（动画）

    vue在插入更新或一处dom时，提供多种不同方式应用过程，本质还是使用css动画，
    基本用法： 使用transition组件，将要执行动画的元素保函在改组件内

    组件

    组件是vue最强大的功能之一，组件可狂战html元素，封装重用代码，组件是自定义元素对象。
